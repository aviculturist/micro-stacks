# Authentication

> Please make sure you've read the quick start guide. Depending on the type of react application, there are some steps you need to do to integrate the `micro-stacks/react` module.

Most if not all Stacks based applications will need to authenticate with a wallet. Currently, the Hiro Web Wallet
extension is the only wallet that can authenticate web based apps. This document will go over how to set up
authentication and different use cases of it.
